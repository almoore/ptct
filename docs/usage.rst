=====
Usage
=====

To use Python Temperature Conversion Test in a project::

  import ptct

To use Python Temperature Conversion Test from the command line. First install
it using the installation instructions

.. code-block:: console

   # When given no response if will return the result
   $ ptct 32 F C
   0
   $ ptct 32 F C --response 0
   correct
